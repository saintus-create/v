<h1 align="center">
  Documentation Shell
</h1>

[![](https://dcbadge.vercel.app/api/server/fdXy3Sk8Gq?style=flat)](https://discord.gg/fdXy3Sk8Gq)

A generic, reusable documentation template built with Svelte and shadcn-svelte components. Use it as a starting point for your own project documentation.

> **Note**
> This repository is a template; customize the content, branding, and configuration to fit your project.

Accessible and customizable components that you can copy and paste into your apps. Free. Open Source. **Use this to build your own component library**.

![hero](docs/static/opengraph-image.png)

## Documentation

Visit the live docs at `http://localhost:5173` after running the dev server, or host it on your preferred platform.
